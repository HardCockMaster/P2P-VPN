Установка P2P VPN
1. Загрузка установщика

Скачайте архив для вашей операционной системы со страницы релизов и распакуйте его. В архиве находятся:

    Исполняемый файл программы (p2p-vpn или p2p-vpn.exe)

    Установочный скрипт (install.sh для Linux/macOS или install.ps1 для Windows)

2. Запуск установки
Linux / macOS

    Откройте терминал и перейдите в папку с распакованными файлами.

    Дайте скрипту права на выполнение:
    bash

    chmod +x install.sh

    Запустите установку:
    bash

    ./install.sh

    При необходимости скрипт запросит пароль sudo для копирования программы в /usr/local/bin и установки специальных прав (capabilities) для создания TUN-интерфейсов.

Windows

    Откройте PowerShell от имени администратора (ПКМ по значку PowerShell → «Запуск от имени администратора»).

    Перейдите в папку с распакованными файлами.

    Выполните команду:
    powershell

    Set-ExecutionPolicy Bypass -Scope Process -Force
    .\install.ps1

    Скрипт скопирует программу в %LOCALAPPDATA%\Programs\p2p-vpn и добавит её в PATH.

    После установки вручную скачайте драйвер Wintun с wintun.net и поместите файл wintun.dll в папку C:\Users\<ВашеИмя>\AppData\Local\Programs\p2p-vpn (туда же, куда установлена программа). Этот драйвер необходим для создания виртуального сетевого адаптера.

3. Запуск программы

После установки просто введите в терминале (командной строке):
bash

p2p-vpn

Если терминал был открыт до установки на Windows, перезапустите его, чтобы обновить переменные окружения.
